Abstract

Hybrid Intelligent Systems Ontology (HISO) is an ontology describing Hybrid Intelligent Systems as systems that combine methods from different AI subfields.

Introduction back to ToC

We conceptualize a Hybrid Intelligent System (HIS) as an AI system that combines methods from different AI subfields and leverages each method’s advantages in order to resolve a problem in a specific application domain. In particular,each method has certain benefits and drawbacks and their combination into an HIS allows the benefits to counteract the drawbacks.
